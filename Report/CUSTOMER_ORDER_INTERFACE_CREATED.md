## 🎨 UI/UX 구현

### 레이아웃 구조

```
헤더
├── "COZY" 브랜드명 (왼쪽)
└── "주문하기", "관리자" 버튼 (오른쪽)

메인 콘텐츠
├── 메뉴 영역 (2/3 너비)
│   └── 메뉴 카드 그리드 (3열)
└── 장바구니 영역 (1/3 너비, 고정)
    ├── 주문 내역 리스트
    ├── 총 금액
    └── 주문하기 버튼
```

### 메뉴 카드 구조

1. **이미지 플레이스홀더**
   - 회색 배경, 대각선 X 표시
   - 48px 높이

2. **메뉴 정보**
   - 메뉴 이름 (굵은 글씨)
   - 가격 (옵션 포함 계산)
   - 간단한 설명

3. **옵션 체크박스**
   - "샷 추가 (+500원)"
   - "시럽 추가 (+0원)"

4. **담기 버튼**
   - 회색 테두리
   - 호버 효과

### 장바구니 구조

1. **제목**: "장바구니"

2. **주문 내역**
   - 메뉴명 (옵션 포함)
   - 수량
   - 가격
   - 삭제 버튼

3. **총 금액**
   - 굵은 글씨
   - 우측 정렬

4. **주문하기 버튼**
   - 전체 너비
   - 회색 테두리

## 📝 주요 기능

### 1. 메뉴 선택

- 메뉴 카드 클릭으로 장바구니 추가
- 옵션 선택 (샷 추가, 시럽 추가)
- 가격 자동 계산

### 2. 옵션 관리

- 체크박스로 옵션 선택
- 옵션 선택 시 가격 실시간 업데이트
- 옵션 정보 장바구니에 포함

### 3. 장바구니 관리

- 선택한 메뉴 표시
- 옵션 정보 포함 표시
- 수량 표시
- 항목별 가격 표시
- 총 금액 계산
- 항목 삭제

### 4. 주문 처리

- "주문하기" 버튼 클릭
- 주문 페이지로 이동 (향후 구현)
- 장바구니 데이터 전달
