## 주요 기능

### 대시보드
- 총 주문 수, 총 매출, 오늘 주문 통계
- 피크 시간대 시각화

### 주문 관리
- 주문 목록 조회
- 상태별 필터링 (전체, 대기중, 준비중, 준비완료, 완료, 취소)
- 주문 상태 변경 (준비 시작, 준비 완료, 완료 처리, 취소)
- 주문 상세 정보 표시

### 메뉴 관리
- 메뉴 목록 조회 (카드 형태)
- 메뉴 추가/수정/삭제
- 상태 관리 (판매중/품절)
- 폼 검증 (React Hook Form + Zod)

## 접근 방법

### 개발 서버 실행
```bash
cd frontend
npm run dev
```

### 관리자 페이지 접근
- 대시보드: `http://localhost:3000/admin`
- 주문 관리: `http://localhost:3000/admin/orders`
- 메뉴 관리: `http://localhost:3000/admin/menus`
