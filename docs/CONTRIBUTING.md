# 기여 가이드

## 개발 환경 설정

1. 저장소 클론
2. 의존성 설치
3. 환경 변수 설정
4. 데이터베이스 마이그레이션
5. 개발 서버 실행

## 코딩 컨벤션

- TypeScript 사용
- ESLint 규칙 준수
- Prettier 포맷팅
- 의미 있는 커밋 메시지

## 브랜치 전략

- `main`: 프로덕션 브랜치
- `develop`: 개발 브랜치
- `feature/*`: 기능 개발 브랜치
- `fix/*`: 버그 수정 브랜치

## Pull Request

1. 기능 개발 완료 후 PR 생성
2. 코드 리뷰 요청
3. CI/CD 통과 확인
4. 승인 후 머지
